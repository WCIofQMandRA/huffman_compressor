中国科大 2021 秋 数据结构（011127.01） 实验 02

Version 1.0.0

## 生成

编译本程序需要 CMake 和支持 C++ 17 的编译器。

```sh
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```

编译的输出位于`build\output`。
